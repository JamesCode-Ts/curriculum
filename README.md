# Curriculum

Currículo profissional desenvolvido utilizando HTML5 e CSS3, com foco em:

- Layout moderno e minimalista
- Compatibilidade com ATS (Applicant Tracking Systems)
- Impressão otimizada para A4
- HTML semântico
- CSS puro (sem frameworks)
- Design responsivo
- Tipografia limpa e hierarquia visual consistente

## Tecnologias

- HTML5
- CSS3

## Objetivo

Este projeto tem como finalidade manter uma versão online e versionada do meu currículo profissional, permitindo sua evolução ao longo do tempo por meio do Git.
